# Diagram System — The Builder's Bible

## Approach
Every chapter gets 2-4 diagrams. Diagrams are embedded as:
1. **Mermaid blocks** in markdown (rendered to SVG at build time)
2. **ASCII art** in code blocks (for terminal-like diagrams)
3. **HTML/CSS diagrams** (for complex visual explanations inline in the HTML)
